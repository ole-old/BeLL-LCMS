A simple HTML app that displays resources in a CouchDB and will HTTP POST them to the existing BeLL LMS's Sync API. 

![Screenshot](screenshot.png)


# Getting Started

Make sure you have Node Couchapp installed. If you don't follow the instructions here -> https://github.com/mikeal/node.couchapp.js

Clone this repository and then 'npm install' in the root directory.  You can then 'couchapp push app.js http://username:password@youserver.com/database'.
